# 1. 项目概述
SQL上线管理程序是面向YashanDB的开发自助+DBA审核平台，提供CLI+Web双端界面，实现SQL提交、自动审核、多级人工审核、上线执行全流程管理。

# 2. 技术选型
- 后端：Go + Gin框架
- 前端：Vue3 + Element Plus
- 存储：YashanDB（存储业务数据）
- 客户端：Go编写CLI工具

# 3. 核心模块
1. **用户权限模块**：支持开发/开发组长/DBA三种角色权限控制
2. **SQL提交模块**：支持Web/CLI双端提交SQL，支持批量提交、上线时间预约
3. **自动审核引擎**：内置YashanDB语法校验、性能检查、安全规则，支持自定义正则规则
4. **审核流程模块**：固定两级审核流程：开发提交→自动预审核→开发组长审核→DBA审核
5. **SQL执行引擎**：支持上线执行、回滚方案生成、执行结果实时反馈
6. **日志审计模块**：全操作日志记录，支持上线历史查询、操作追溯

# 4. 核心流程
```
开发提交SQL → 系统自动预审核（不通过直接驳回） → 开发组长审核（驳回/通过） → DBA审核（驳回/执行/预约执行） → 执行SQL → 结果通知提交人
```

# 5. 特性支持
- 仅支持YashanDB数据库
- CLI+Web双端操作
- 全量审核规则+自定义规则支持
- 固定两级审核流程
- 上线回滚方案自动生成
- 操作全链路审计
